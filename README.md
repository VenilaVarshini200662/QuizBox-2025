DESCRIPTION:

  QuizBox is an interactive quiz Application where users can answer questions and receive instant feedback.
  Its features are,
    *Dynamic question loading and scoring.
    *Category Selection
    *Correct answer is displayed intantly for each wrong answer.
    *Feedback can be send.
    
  TOOLS AND TECHNOLOGIES USED:
    * VScode.
    * HTML,CSS,Javascript and Emailjs to ensure realtime feedback facility.
